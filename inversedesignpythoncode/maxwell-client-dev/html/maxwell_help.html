Introduction to Maxwell

Along with the functionality of Maxwell, this library includes various examples which are useful in familiarizing oneself with Maxwell.

Maxwell functions

  • maxwell_axb - Matrices and vectors associated with the electromagnetic wave equation.
  • maxwell_box - Box of constant epsilon/mu within the simulation grid.
  • maxwell_box_smooth - Box of constant epsilon/mu within the simulation grid with smoothed boundaries.
  • maxwell_cyl - Cylinder of constant epsilon/mu within the simulation grid.
  • maxwell_cyl_smooth - Cylinder of constant epsilon/mu within the simulation grid with smoothed boundaries.
  • maxwell_flux - Electromagnetic power passing through a finite plane of the simulation.
  • maxwell_fsmode - Excitation source for arbitrary free-space modes.
  • maxwell_gaussian - Excitation source for free-space Gaussian modes.
  • maxwell_grid - Initialize a simulation domain.
  • maxwell_pos2ind - Translate from position to array indices.
  • maxwell_shape - Add shapes of constant material to the simulation domain.
  • maxwell_solve - Solve the electromagnetic wave equation.
  • maxwell_solve_async - Electromagnetic solve without waiting for completion.
  • maxwell_solve_eigenmode - Eigenmode based on an initial guess for E.
  • maxwell_view - View a slice of the field and/or structure.
  • maxwell_wgmode - Excite a waveguide mode. Can also used to filter for a particular mode.

Examples

  • example0_waveguide - Excite the fundamental mode of a waveguide and measure output power.
  • example1_ring - Excite the clock-wise mode of a ring resonator.
  • example2_metalmode - Solve for the resonant mode of a metallic resonator.
  • example3_cavitymode - Solve for the eigenmodes of dielectric resonators.
  • example4_focusedbeam - Excite Gaussian or donut free-space modes.
  • example5_hiresmetal - Solve for the resonant mode of a metallic resonator.

Introduction to Maxopt

The maxopt family of functions, examples, and cases lives on top of the maxwell functions and implements basic structural optimization functions and examples.

Maxopt functions

These functions enable gradient-based optimization schemes.

  • maxopt_field_gradient - Calculate structural gradients for the E-field of a simulation.
  • maxopt_freq_gradient - Calculate structural gradients for frequency of the eigenmode.
  • maxopt_gradient_descent - Simple gradient descent optimization algorithm.
  • maxopt_solve_adjoint - Asynchronously solve for the adjoint of the operator.

Maxopt examples

Various optimization examples. The various optimization methods showcased include brute-force parameter sweep, derivative-free search, and gradient-descent.

  • maxopt_example0_sweep - Sweep the parameters of a waveguide-coupled disk resonator structure.
  • maxopt_example1_search - Derivative-free optimization of a nanophotonic grating coupler.
  • maxopt_example2_adjoint - Form a cavity out of a square lattice using gradient optimization.
  • maxopt_example3_eigenmode - Derivative-based optimization of an L3 cavity mode.

Maxopt cases

These cases are used in the Maxopt examples and show how structures can be set up for optimization.

  • maxopt_case_2wbeam - Sets up a frequency-doubling cavity optimization.
  • maxopt_case_L3 - Sets up the optimization for an L3 photonic crystal resonator.
  • maxopt_case_grating - Sets up a grating coupler optimization problem.
  • maxopt_case_metalfocus - Used to optimize a metal focusing structure.
  • maxopt_case_squarepc - Sets up optimization problem for a square photonic crystal array.
  • maxopt_case_wdmgrating - Used to optimize a wavelength-splitting grating coupler.
